# Mobile Sales Performance Dashboard 📊

An advanced, interactive multi-page Power BI dashboard designed to analyze retail mobile sales data, track dynamic time-intelligence growth metrics, and discover consumer behavior patterns.

## 📌 Project Overview
This project transforms raw transactional retail data into actionable business intelligence. It provides key stakeholders with micro and macro-level trends regarding brand performance, regional demands, payment preferences, and cumulative period growth.

## 🛠️ Key Technical Features Implemented
- **Advanced Data Modeling:** Built a robust data model using a custom calendar date dimension linked via a 1-to-many relationship to the main sales table.
- **Time Intelligence Analysis (YoY):** Formulated complex DAX calculations to track dynamic sales variables against the **Same Period Last Year (SPLY)** across year, quarter, and month hierarchies.
- **Month-to-Date (MTD) Metrics:** Developed a dedicated cumulative growth line view mapped out by daily progression logs.
- **Interactive ETL & Cleansing:** Handled calendar string formatting and dimensional sequence sorting using customized `MonthNum` parameters.

## 📊 Business Metrics Discovered
- **Total Revenue Managed:** $769M 
- **Total Volume Exceeded:** 19K Units Sold
- **Total Transactions Registered:** 4K Logs
- **Top Consumer Segment Channels:** UPI & Credit Cards dominate the transaction volume share.
